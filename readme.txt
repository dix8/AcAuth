使用说明：
请使用 创建数据表.txt 文件中的命令创建数据表或导入sql文件使用 请在config.php中修改key

各api使用教程：
check.php查询
返回值：1 已授权 0 未授权
使用方法：yourdomain.com/check.php?content=参数

insert.php写入数据
使用方法：yourdomain.com/insert.php?key=你的key&qq=QQ号码content=参数

delete.php删除数据
使用方法：yourdomain.com/insert.php?key=你的key&qq=QQ号码content=参数

update.php修改数据
使用方法：yourdomain.com/update.php?key=你的key&qq=QQ号码content=参数&new_content=参数

data.php查课所有数据
使用方法：yourdomain.com/data.php?key=你的key

list.php查课所有数据
使用方法：yourdomain.com/list.php
直接访问即可显示所有数据，跟data.php一样，无需输key版
